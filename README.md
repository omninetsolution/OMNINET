# OMNINET SOLUTIONS

Independent third-party service assistance platform for internet, cable TV, and streaming connectivity guidance.

## Project Overview

OMNINET SOLUTIONS is a Google Ads policy-compliant website launched in 2026. We provide independent guidance and assistance services to help customers navigate connectivity options.

## Technologies Used

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting Started

Follow these steps to run the project locally:

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory
cd <YOUR_PROJECT_NAME>

# Step 3: Install dependencies
npm install

# Step 4: Start the development server
npm run dev
```

## Development

- Run `npm run dev` to start the development server
- Run `npm run build` to create a production build
- Run `npm run preview` to preview the production build locally

## Compliance

This website is designed to be fully compliant with Google Ads policies:
- Independent third-party disclosure on all pages
- No provider impersonation
- Clear refund policy (7-15 days)
- Mandatory FAQ compliance questions
- Separate service fee disclosure
